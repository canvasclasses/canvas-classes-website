# Chapter 13 review — अपठित बोध — पठन अभ्यास

SUMMARY: This is an exam-prep module (CBSE Section A practice), not a literature chapter, so the review is re-scoped per instructions: factual self-consistency of each passage + answerability-from-passage-alone + assessment integrity + natural Hindi. The teaching frame (4-step strategy, friendly voice) is sound and the passages are mostly self-contained. But the chapter ships **one CRITICAL arithmetic contradiction that is baked into both a passage and a graded assertion-reason item** (School स: 220 planted, 110 survived = *exactly* half, yet the text and the question both assert "not even half survived"), which mis-marks any student who reasons correctly. The single biggest systemic weakness is **assessment fragility on the case-based/data page**: the one page that introduces numbers is also the one page whose numbers don't support the claims built on them, and a second item there is answerable by elimination. Secondary weakness: cross-page formulaic uniformity — every passage is the same "small-effort-compounds / perseverance" homily, and the MCQ shape (two recall + one शब्दार्थ + one AR + one short-answer) is stamped identically across all five passages.

[CRITICAL] Page 2 — School स 220 planted / 110 survived is **exactly half**, but both the passage ("आधे भी न बच सके") and assertion-reason Q4's assertion ("आधे पौधे भी जीवित न रह सके") state *fewer than half* survived. The number contradicts the prose; the AR explanation even endorses the false assertion as "सही". Mis-marks a numerate student.
2. [MAJOR] Page 2 Q4 (assertion-reason) — the assertion is itself factually false (see #1), so the keyed answer (option 2 = "A सही, R ग़लत") is wrong on the A-half; a correct reading gives "A ग़लत".

4. [MAJOR] Cross-page uniformity — 3 of 5 passages ("छोटी आदतें", "तीन स्कूलों…", "समय सबसे बड़ा धन") deliver the identical thesis (small/steady effort compounds via a banking/interest or habit analogy), and the explicit cross-passage link ("ठीक वैसे ही जैसे कोई अच्छी आदत") makes passage 2 derivable from passage 1. [MINOR→MAJOR] Length-tell drift on the longer-stem MCQs (Page 1 Q5, Page 4 Q4): the correct interpretive option is the fullest, most hedged sentence while at least one distractor is a short throwaway — the "correct-is-longest" pattern the workflow §5.12.1 specifically warns about, now leaking into `reading_comprehension.questions`.
