## B. Must-memorize relations he teaches (use in ⚡ sections)

- **Rate definition:** rate = ±(1/stoich coeff)·d[X]/dt — minus on reactants "taaki positive aa jaye; rate has to be positive, it is speed." The 1/coefficient is derived live, never memorised.
- **Units of k:** (concentration)^(1−n) per time. **His detective move:** "Question mein order nahi bataya? Focus on K ke units kya diye hain — usi se order pata lag jayega." (mol/L/time → zero, /time → first, L/mol/time → second)
- **Zero order:** Ct = C0 − kt (straight line, slope −k, intercept C0); x = kt (linear in t); rate = k throughout; t_completion = C0/k; **t½ = C0/2k — directly proportional to initial concentration** ("initial concentration double karoge toh t-half bhi double").
- **Zero-order shortcut (he names it):** *equal concentration drop in equal time intervals* — solve orally, no formula.
- Zero-order examples to know: NH3 → N2 + H2 on Pt/W surface (reverse Haber), H2 + Cl2 → 2HCl (photochemical), 2HI → H2 + I2 on gold, 2N2O → 2N2 + O2 on platinum. "Thoda sa padh loge toh kaam aa jata hai."
- **Law of mass action vs rate law:** mass action = powers from stoichiometry (works only for elementary steps, where molecularity = order); **rate law = powers from EXPERIMENT** — "tum shakl dekh ke nahi bata sakte. Sir kaise pata lagega? Beta, experiment karo."
- Order can be positive, negative, zero, fractional — "usko koi samasya nahi hai." Molecularity = total particles in an elementary step.
- Graph-reading move he drills: stoichiometry ⇄ concentration-time graph. Drop in A vs gain in B measured in *blocks* on the graph — equal blocks ⇒ 1:1; double gain ⇒ coefficient 2 (his H2O2 → H2O + O2 example: water's gain ≈ 2× oxygen's ⇒ 2H2O2 → 2H2O + O2).
- **First order — THE formula:** k = (2.303/t)·log(C0/Ct) = (2.303/t)·log(a/(a−x)) — "ye formula hi aapko throughout first order mein kaam aayega. Achhe se yaad hona chahiye." Ct = C0·e^(−kt) (exponential drop; rate-vs-t and conc-vs-t graphs look the same).
- **First order t½ = 0.693/k — INDEPENDENT of C0.** His t½-vs-order ladder: zero order ∝ C0, first order independent, second order ∝ 1/C0, third ∝ 1/C0². "Relation establish kar sakte hain."
- **After n half-lives:** Ct = C0/2ⁿ. And **t75% = 2·t½** — "iski kahani badi interesting hai" (log 4 = 2 log 2, derive it, don't ratta).
- **First order never completes** — "t-completion nahi hota usmein. Infinite time lagega." (CT→0 ⇒ t→∞.)
- First-order examples to know: N2O5 → NO2 + O2 ("kaafi zyada questions mein milega"), radioactive decay, H2O2 decomposition, acidic ester hydrolysis, sucrose inversion (→ glucose + fructose), NH4NO2 → N2 + 2H2O. "Examiner kabhi-kabhi expect karta hai ki kuch cheezein yaad bhi honi chahiye."
- **Two-observation case (C0 not given):** k = 2.303/(t2−t1)·log(C1/C2) — "jahan se TUMNE observe karna start kiya wahi tumhara C-naught hai. You don't always have to know the actual initial concentration."
- **Gas-phase first order:** NO pre-made formula — "gaseous ke case mein humein har case mein formula on-the-spot derive karke solve karna padta hai." Build the pressure table (P0 / P0−P / P / P...), express P from Ptotal, and remember: a/(a−x) = initial/final pressure **of the reactant**, never total.
- **Ester-hydrolysis titration setup:** V0 ∝ catalyst acid (b); Vt ∝ b + x; V∞ ∝ a + b. So a = V∞ − V0 and a − x = V∞ − Vt. "Agar formula nahi pata toh values de ke bhi kya fayda? Aur ratta hua hai par samajh nahi — toh samajhna hai ki formula aaya kaise. Phir aage galti nahi hogi."
- **Parallel reactions (same order paths):** K_overall = K1 + K2; **[B]/[C] = K1/K2 at any time** (the (a−x) cancels). Opposing: net rate = k1(a−x) − k2x; at equilibrium k1(a−x) = k2x. Steady state (d[B]/dt = 0): K1[A] = K2[B] ⇒ [B] = (K1/K2)[A].
- **Degree of dissociation, first order:** α = 1 − e^(−kt) (derive from Ct = C0e^(−kt) — "alpha is amount dissociated upon initial").
- **Arrhenius:** k = A·e^(−Ea/RT). The exponential term = "fraction of collisions with the minimum energy for the reaction" — that fraction → 1 when Ea = 0 or T → ∞, and then k = A. A (pre-exponential) = "measure of the rate at which collisions occur, irrespective of their energy."
- **The two famous Arrhenius graphs (both asked by IIT and Mains):** ln k vs 1/T → straight line, slope −Ea/R (negative); k vs T → exponential increase. A and Ea treated as temperature-independent ("itna drastic change nahi karne wale ki in dono ki value badle").
- **Two-temperature workhorse:** ln(k2/k1) = (Ea/R)·(1/T1 − 1/T2). "Kitne times rate badh gaya? Matlab k2/k1 hi nikalna hai."
- **Catalyst speed-up factor:** k_cat/k_uncat = e^((Ea − Ea′)/RT) (A cancels — same reaction, same T).
- **Catalyst-vs-temperature equal-rate condition:** A·e^(−Ea′/RT1) = A·e^(−Ea/RT2) ⇒ **Ea′/T1 = Ea/T2**.
- **Composite rate constants:** if K = k1k2/k3 then A = A1A2/A3 and **Ea(overall) = E1 + E2 − E3** (exponents add/subtract like the k's multiply/divide).
- Catalyst facts he lists: provides lower-Ea alternate path; does NOT change ΔH/ΔG or equilibrium position ("thermodynamics ko disturb nahi karta"); does NOT initiate a non-occurring reaction; small amount suffices ("kilo ke bhaav, quintal ke bhaav nahi daalna"). Positive/negative/auto-catalyst classification; KClO3 + MnO2 standard example.
- **Radioactivity = first order with renamed parts:** rate = λN; N = N0·e^(−λt); t½ = 0.693/λ; **average life τ = 1/λ = 1.44·t½**; λ = (2.303/t)·log(N0/N). Weight ∝ number of nuclei (w/at.wt = moles) — so weight ratios can replace N ratios directly.
- Activity units: **1 Curie = 3.7×10¹⁰ dps; 1 Rutherford = 10⁶ dps.**
- U-238 → Pb-206: 8 alpha + 6 beta — balance BOTH mass number (32/4 = 8α) and atomic number (−16 + 6 = −10). "Dono properly match karne chahiye."

## C. Trap bank (concept → the reflex he dramatizes, for ⚠️ sections)

- **Grams given, mol/L needed (his flagged "silly mistake"):** rate of SO3 formation = 100 g/min, asked dO2/dt in g/min. "Maximum bachche kya galti karte hain? Bas equation lagate hain, 100 put karte hain, divide by 2, 50 bolte hain — sir answer aa gaya 50." Convert 100 g → 100/80 mol FIRST, do the rate ratio, convert back ×32 → 20 g/min. "Pooch kya raha hai? Units ka dhyaan rakhein."
- **Rate vs order sign confusion:** "Rate of reaction can be positive, negative, zero" — FALSE. "Rate negative nahi ho sakta, it has to be positive. Haan, aapka ORDER positive, negative, zero ho sakta hai. Rate nahi."
- **Units-of-rate vs units-of-k:** units of rate of reaction do NOT depend on order (always conc/time); it's the rate CONSTANT whose units shift with order.
- **Species in the denominator of a rate law:** rate = k[I⁻][OCl⁻]/[OH⁻] — "aise ghabrana nahi hai ki sir isko neeche likh diya toh kya hoga. Usko upar le jao — raise to power −1." Overall order = 1+1−1 = 1.
- **"Rate constant for first order, so rate is constant" reflex:** No — rate = k[A]¹ drops as concentration drops. Only zero order has constant rate.
- **Kf = Kb at equilibrium because moles are equal?** "Equal moles of reactant and product — achhi baat hai. But do we know order kya tha? Nahi. Toh kaise predict karenge?" Only true if both orders are 1.
- **Mechanism questions — intermediate in the slow step:** if the RDS contains a species (HS⁻) generated in a prior fast equilibrium, substitute its concentration from K_eq — the rate picks up extra dependence (here 1/[H⁺]) and may no longer match the given law. Slow step decides; molecularity of slow step = order ONLY for the species as they appear there.
- **Law of mass action over-trust:** stoichiometric coefficients ≠ order for multi-step (complex) reactions. "Isliye law of mass action galat tha."
- **Rate law isn't fixed for a reaction:** H2+Br2 rate law changes form at the very beginning vs very end of the reaction (ignore small terms); NH3 on tungsten is ~zero order when NH3 is high, ~first order when low. Mechanism (and so the rate law) can change with T, P, concentration.
- **Rate law can contain products and catalysts:** 2O3 → 3O2 depends on O2 (product); NO (catalyst) appears in rate laws. "Aisa nahi ki hamesha reactant hi honge."
- **Blind-calculation reflex (his named warning):** the C2H4O gas question — students find P = 40 and dive into k = 2.303/t·log(80/40). "Aage calculate karne mein mat lag jaana — 80 se exactly half ho gaya 20 minute mein. Ye 20 minute hi toh t-half hue na! Aankhein khuli rakhna; andhe ki tarah mat daud jaana."
- **"First order completes in X seconds" statement:** auto-false — first order never completes. He kills it first in every multi-statement question.
- **Total pressure in the first-order formula:** "P-naught aur P-total ki jagah pe NAHI — initial pressure of reactant upon final pressure of reactant. Reactant ke respect mein hi ye derive hua hai poora."
- **Time limits that don't start at zero (bacterial-growth PYQ):** the antibiotic takes 1 hour to reach the wound — integrate time from 1 to t, NOT 0 to t, and the decaying population starts at n0·e, not n0. "Galtiyan hone ka scope kahan-kahan pe tha? Time humein zero se t nahi jaana hai."
- **ln(P/P0) vs ln(P0/P):** first-order linearity is in ln(C0/Ct) vs t. If the plot's axis is inverted (P/P0), it's NOT a straight line with positive slope — "ulta natural log of P0/P hota toh direct relation aa jata."
- **Factor-of-2 in elementary 2A ⇌ A2:** asked d[A]/dt, students write k1[A2] − k−1[A]² and "ye 1/2 wala part chhod dete hain." Correct: d[A]/dt = 2k1[A2] − 2k−1[A]². "Two wala part tha usi ke chakkar mein galti ho jati hai."
- **t1/2-proportionality as an order detector (sugar-inversion Q):** t½ same "for any concentration of sugar" ⇒ first order in sugar; t½ drops 10× when [H+] drops 10× (pH 5→6) ⇒ t½ ∝ [H+] ⇒ zero order in H+. "Koi numerical solve hone wala nahi tha — data ko samajhte hue proportionality nikalni thi."
- **t10%/t90% folklore number:** the ratio is log(100/90)/log(10) ≈ 0.046/... — he computes t90/t10 ≈ 22, NOT the commonly-quoted 25. "Maine calculate karke dekha tha — 22 ke aas-paas aa raha hai. Aap ek baar khud se bhi solve karke dekh lena." (Models honest verification.)
- **"High Ea ⇒ fast reaction" reflex:** "Arre bhaiya, pahadi zyada oonchi ho jayegi toh jaldi chadh paoge kya? Nahi — aur der hogi." High Ea = higher energy demand = SLOWER.
- **Energy alone isn't enough (collision theory):** sufficient energy AND correct orientation both needed — "sirf energy poori honi zaroori nahi hai." His funnel numbers make it concrete (100 → 20 → 10).
- **Scary-looking composite mechanisms:** A+B ⇌ C then C → P with K = k1k2/k3 — "dekhne mein khatarnak lagta hai, thoda darawna lagta hai... par iska solution actually chhota tha." Write each k as Arrhenius, multiply/divide, match exponents.
- **Removed-syllabus jargon panic (chemisorption):** "NTA ne syllabus se chapters hata diye, par naam aa hi gaya. Pareshaan nahi hona hai — wo ek tarah ka adsorption hi hai. Aake chipak raha hai, bas. Humein sirf rate ka ratio lena hai." Don't let an unfamiliar word block a standard calculation.
- **Given constants override habit:** in the adsorption question R is given as 8.3 and ln 10 as 2.3 — so the answer uses 2.3, not 2.303. "Kyon 2.3? Question mein diya hua hai — 8.303 nahi, 8.3 hi diya hai." Read what the paper gives you.
- **k = time confusion (milk question):** he explicitly disclaims it — "main ye NAHI bol raha k = time. Jo ratio time ki aayegi, wo k ki ratio ko represent karegi" (k2/k1 = t1/t2).
- **Kinetic product folklore:** "kinetic product hamesha higher energy mein hota hai — aisa guarantee ke saath nahi bol sakte." It's defined by forming FASTER, not by energy.
- **Useless given data:** the Sr-90 bones question — "pehli statement kisi kaam ki nahi hai humari" — he openly discards flavor text and pulls the two numbers that matter (t½, w0).

## D. Per-question exemplars (worked Qs, opening → key move → closing)

### From L1 (rate, rate law, zero order)

1. **log(−da/dt) = log(db/dt) + 0.3, xA → yB, find x:y** — Open: "Pause the video here, 1 minute ke andar solve karna. Aasan question hai." Key: write rate = −(1/x)da/dt = (1/y)db/dt, rearrange, take log of both sides, match: log(x/y) = 0.3 = log 2 ⇒ x/y = 2. Close: "x = 2, y = 1 is one of the possibilities... Simple question."
2. **Rate = k[I⁻][OCl⁻]/[OH⁻], overall order?** — Key: denominator → power −1; 1+1−1 = 1. Close: "Aise ghabrana nahi hai... overall order aapka first order reaction hai ye."
3. **SO3 forming at 100 g/min; O2 disappearance in g/min?** — Open: "1 minute ke andar-andar solve ho jana chahiye. Dhyaan se padhna — ismein galti hoti hai, silly mistake hoti hai." Key: dc/dt lives in mol/L/time; 100 g → 100/80 mol, ratio ½, back ×32 ⇒ 20 g/min. Close: names the mass-error reflex ("50 bolte hain sir answer aa gaya") + "units ka dhyaan rakhein."
4. **−dPA/dt = k′PA² and −(1/V)dnA/dt = kCA² — relate k′ and k** — Key: show −(1/V)dn/dt IS −dc/dt ("volume ko andar le lo — number of moles upon volume kya ho jata hai? Concentration"); P = CRT ⇒ k′(RT)² = k... structure: both expressions are the same rate in different clothes.
5. **Which reaction's rate can be measured easily?** — Open: "Thoda sa common sense laga ke sochna — not ki aapne ye directly cheez padhi hui hai." Key: AgNO3 + KI ionic precipitation = very fast; rusting = very slow ("kaun baitha rahega wahan pe?"); N2+3H2 → NH3 neither too fast nor too slow ⇒ measurable. 
6. **Concentration-time graph → stoichiometry** — Open: "30 seconds. Fast. Bas dekhne ke liye ki aap stoichiometry aur graph ko relate kar pa rahe ho ki nahi." Key: count drops/gains in graph blocks — A drops 2, B gains 1, C gains 3 ⇒ 2A → B + 3C. Close: "Simple. Very simple."
7. **True statements about rate (multi-statement)** — Key: kills "rate can be negative" (order can, rate can't); kills "units of rate depend on order" (that's k); accepts "average = instantaneous possible" (zero order straight line, ya interval chosen so they match). Close: "Only C option is correct."
8. **Zero order, [A]: 1 → 0.8 in 10 min; t½?** — Open: "Shortcut se solve hoga. 30 seconds. Fast." Key: equal drop in equal time — 0.2 per 10 min ⇒ five intervals to zero ⇒ t_completion 50 min ⇒ t½ = 25 min. Close: "Koi formula mujhe lagana nahi pada — orally solve kar diya. And it's not a shortcut ki humne kuch jugaad lagaya hai. Humne equation ko samjha hai."
9. **Cl2O7 → 2ClO2 + 3/2 O2 (zero order, −dP/dt = 50 mmHg/s, P0 = 600), four parts** — Open: "1 minute. Iske chaar parts hain — chaaron ko solve karna hai." Key: dO2/dt = (3/2)·50 = 75; completion 600/50 = 12 s ⇒ t½ = 6 s ("ye common sense hai, directly solve ho gaya"); rate at t½ = k = 50 (zero order, rate constant throughout); total P after 10 s via the P-table: 600−500=100, 2P → 1000, (3/2)P → 750 ⇒ 1850 mmHg. Close: "Samajh gaye na ye cheez? Samjhe ki nahi samjhe?"
10. **Kf vs Kb for equilibrium with equal moles** — Open: "30 seconds for this one. Not difficult. Aasan question hai ye." Key: Rf = Rb at equilibrium, but powers (orders) unknown ⇒ can't claim Kf = Kb. Close: dramatizes the gap — "x aur y nahi pata humein. Order of reaction kya hai? Ye nahi pata."
11. **Cl2 + H2S mechanism consistency (rate = k[Cl2][H2S])** — Open: "Dhyaan dena bete." Key: RDS rule — mechanism A's slow step has Cl2 + H2S directly ⇒ matches; mechanism B's slow step uses HS⁻ (an intermediate) ⇒ substitute from the fast equilibrium ⇒ rate ∝ [Cl2][H2S]/[H⁺] ⇒ extra H⁺ dependence ⇒ doesn't match. Close: "Only A. Baaki chinta nahi karo — is tarah ke question aage aur bhi aayenge."

### From L2 (first order, gas phase, pseudo-first-order, parallel/opposing/consecutive)

12. **First order, 1 → 0.6 in 20 min; time for 0.6 → 0.36?** — Open: "30 seconds... chalo 45 laga lo, par achhe se solve karo." Key: equal-percentage shortcut — 40% drop in 20 min ⇒ 0.6 ka 40% (0.24) goes in the NEXT 20 min ⇒ 0.36. Then shows the proper method too (two k-expressions, equate, 2.303 cancels). Close: "Shortcut kab applicable hoga, kab nahi? It is ALWAYS applicable, because it is not a shortcut — it is zyada aasaan understanding of the concept. Formula hi lagana zaroori nahi — dimaag bhi laga sakte hain kabhi-kabhi."
13. **C4H8 → 2C2H4, when is molar ratio 1?** — Open: "1 minute for this question." Key: order not stated ⇒ "I will look at the units" (per second ⇒ first order); table a−x = 2x ⇒ a = 3x ⇒ t = (2.303/k)·log(3/2). Close: "Bas calculation wala part hai — thodi si log calculations."
14. **Cane sugar inversion: t½ = 500 min at pH 5 for ANY sugar concentration; at pH 6 it becomes 50 min** — Key: two proportionality readings — t½ independent of [sugar] ⇒ first order in sugar ("bachche, t-half reactant ki concentration se independent kab hota hai? First order mein"); [H+] tenth ⇒ t½ tenth ⇒ t½ ∝ [H+] ⇒ zero order in H+. Close: "Koi numerical solve hone wala nahi tha. Data ko samajhte hue — kiski kiske saath proportionality kaisi aa rahi hai."
15. **Gas case template A(g) → B(g) + C(g)** — Key: pressure table P0 → (P0−P, P, P) ⇒ Ptotal = P0 + P ⇒ P = Ptotal − P0; k = (2.303/t)·log[P0/(2P0 − Pt)]. Close: "Stoichiometry change karoge toh formula change ho jayega — isliye har case mein on-the-spot derive karna padta hai. Ek baar samajh aa jaye toh galti nahi hoti. Ek-do baar khud se kar lena."
16. **C2H4O → CH4 + CO, P0 = 80 mm, Ptotal after 20 min = 120; find t½** — Open: "Aasaan hai, 1 minute. Common sense pe focus karna — aisa nahi ki andhe ki tarah calculation karne pe vyast hain aur hints pe dhyaan hi nahi diya." Key: 80 + P = 120 ⇒ P = 40 ⇒ reactant 80 → 40 = EXACTLY half in 20 min ⇒ t½ = 20 min, no formula. Close: "Arre bhai, wahi toh hai — directly dekh lo na. Isliye maine kaha tha aankhein khuli rakhna."
17. **H+ in a 5-mL drop (3×10⁻⁶ mol), k = 1×10⁻⁷ mol/L/s; time to disappear** — Key: units ⇒ zero order ⇒ C0 = kt; concentration = moles/volume with the litre conversion (5 mL → 5×10⁻³ L). Close: "Bas itna hi tha. That's it. Simple question."
18. **2A → 4B + C (first order by units), P0(A) = 1 atm... total pressure 0.15 atm case** — Open: "Likhiye, dimaag — gaseous case hai. Formula aapko derive karna padega; pehle se bana-banaya nahi milega." Key: table 0.1−2p / 4p / p ⇒ Ptotal = 0.1 + 3p; set = 0.15 ⇒ 3p = 0.05; then t = (2.303/k)·log[0.1/(0.1−2p)]. Close: "Always remember — initial pressure of reactant upon final pressure of reactant."
19. **[A] = 4[B], t½(A) = 5 min, t½(B) = 15 min; when equal?** — Open: "I want to see kitne tareekon se solve karte ho — shortcut se, detailed se. Dono se karna aana chahiye." Key: halving chains — A: 4B→2B→B→B/2 (15 min), B: B→B/2 (15 min) ⇒ equal at 15 min. Close: "Laga na shortcut first-order wala — equal percentage drop in equal interval of time."
20. **Bacterial infection PYQ (n = n0·e^t for 1 hr, then dn/dt = −5n²; plot n0/n vs t)** — Open: "Pehle ek-do minute sochiye aaraam se. Thoda tedha question hai — ek nayi si situation hai aapke liye." Key: medicine takes 1 hour ⇒ growth reaches n0·e FIRST; integrate dn/n² with time limits 1 → t and initial conc n0·e; multiply through by n0 ⇒ straight line, positive slope 5n0. Close: itemizes where marks die — "galtiyan hone ka scope: time zero se t nahi, one se t; aur initial concentration n0 nahi, n0·e."
21. **3° halide + strong base substitution, which plots correct (IIT, multi-option)** — Key: chemistry first — "3° halide SN2 dete hi nahi, SN1 hi option hai" ⇒ first-order kinetics ⇒ t½ independent of C0 (flat — correct); initial rate flat? "Arre bhaiya, rate flat kaise ho jayega — concentration drop karegi toh rate bhi drop karega"; Q/P0 exponential rise? "P0 is only a constant — product aise grow nahi karta"; ln(P/P0) vs t straight with positive slope? Inverted ratio — wrong. Close: "Only one option is correct — A."
22. **t(1/8) ÷ t(1/10) × 10 = ?** — Open: "1 minute. IIT ka PYQ hai. Aasaan hai — is chapter se bahut tough questions generally nahi pooche gaye." Key: t(1/8) ∝ log 8 = 3 log 2 = 0.9; t(1/10) ∝ log 10 = 1; ratio 0.9 ×10 ⇒ 9. Close: "IIT ka PYQ hai par phir bhi aasaan tha."
23. **Multi-statement on a first-order k** — Key: "reaction completes in 1000 s" — auto-kill ("first order kabhi complete hota hi nahi"); t½ = 0.693/k check by computation; t10 vs t90 ratio computed honestly (≈22, not the option's 25 — "aap ek baar khud se solve karke dekh lena"); α = 1 − e^(−kt) — TRUE, derived on the spot; "rate and rate constant same units" — false for first order.
24. **A, B equimolar, t½ 54 min and 18 min; when is [A] = 16[B]?** — Key: both methods again. Shortcut: in 54 min B does three halvings for A's one; track the chain — A at 1/4 when B at 1/64 ⇒ 1/4 = 16×(1/64) ⇒ 108 min. Close: "Generally maine dekha hai ki shortcut wala kaam kar jata hai — values aisi di hoti hain ki wo shortcut mein fit ho jaati hain."
25. **Elementary 2A ⇌ A2, expression for d[A]/dt** — Open: "Ye dhyaan dena — ELEMENTARY hai, toh stoichiometry hi order hai." Key: net rate = k1[A2] − k−1[A]²; but rate = (1/2)·d[A]/dt ⇒ d[A]/dt = 2k1[A2] − 2k−1[A]². Close: names the trap — "bas yahan pe galti sirf itni si hoti hai: 1/2 wala part chhod dete hain."
26. **A →(K1) B →(K2) C with d[B]/dt = 0** — Key: steady state in one breath — "kis speed se ban raha hai bhaiya? Zero. Jitna aaya utna gaya." k1[A] = k2[B] ⇒ [B] = (K1/K2)[A].

### From L3 (Arrhenius, collision theory, catalysis, kinetic/thermodynamic product, radioactivity)

27. **Which plot follows the Arrhenius equation (k vs T)?** — Key: instant recall — "abhi-abhi kiya hai, ismein koi dimaag lagane wala kaam bhi nahi" — k vs T is the exponential rise (and ln k vs 1/T the negative-slope line). Both graphs "kaafi popular" with IIT and Mains.
28. **Arrhenius multi-statement (IIT PYQ)** — Open: "Pyaara question hai. Theoretical hai — 1 minute ke andar aaraam se soch lijiye. More than one option correct ho sakti hai." Key: (a) high Ea ⇒ fast? — pahadi-height kill; (b) k rises with T — yes, that IS Arrhenius; (c) due to more collisions exceeding Ea — the extra shaded Maxwell area; (d) higher Ea ⇒ stronger T-dependence — the trekker-tools analogy; (e) A measures collision rate irrespective of energy — yes, energy lives in e^(−Ea/RT). Close: "B, C, D — all three are correct."
29. **Hydrogenation: 500 K uncatalyzed = 400 K catalyzed (same rate), catalyst lowers barrier by 20 kJ; find Ea** — Open: "Aasaan question hai. 1 minute. Dhyaan se karna, jaldbaazi nahi karni." Key: equal rates ⇒ Ea′/T1 = Ea/T2 with Ea′ = Ea − 20 ⇒ one variable, solve. 
30. **H2 adsorption on Pt (Ea = 30) vs Ni (41.4); log of rate ratio** — Open: "Pehle thoda dimaag lagao; nahi lag raha toh phir dekho." Key: k2/k1 = e^((Ea1−Ea2)/RT), take log with the GIVEN constants (R = 8.3, ln 10 = 2.3) ⇒ 1.9 ≈ 2. Close: "Question mein bas chemisorption wala part pareshaan kar sakta tha — normally socho, aake chipak raha hai bas."
31. **Milk splits in 60 min at 300 K, 40 min at 400 K; find Ea** — Key: the time-is-speed reframe — 60/40 IS k2/k1; plug into ln(k2/k1) = Ea/R·(1/T1 − 1/T2). Close: "Interesting part seekhne wali baat: time diya tha, par wo speed ka idea de raha hai. Humne k = time nahi liya — ratio of k ko time ki ratio se relate kiya."
32. **A+B ⇌ C (E1 = 300, E2 = 200), C → P; K = k1k2/k3, overall Ea = 400; find E3** — Open: "Dekhne mein khatarnak lagta hai, thoda darawna." Key: write every k as A·e^(−E/RT); exponents combine ⇒ Ea = E1 + E2 − E3 ⇒ 400 = 300 + 200 − E3 ⇒ E3 = 100 kJ/mol. Close: "Question dekhne mein bada lag raha hai, par solution actually chhota tha."
33. **Enthalpy plot, A + B → C and D; identify the INCORRECT statement** — Key: prune the diagram first ("C ke aage jaane ki zaroorat nahi — faltu mein diya hua hai"); D = kinetic product (smaller hill), C = thermodynamic (lower energy); backward C → A+B has the highest Ea; the last option contradicts the picture — "humein values chahiye hi nahi. Arre bhaiya, D ke liye Ea kam hai, C ki zyada — statement hi apne aap mein galat hai."
34. **Two isotopes, equal initial weights; after 20 days wA/wB = 4:1; t½(A) = 1 day; find t½(B)** — Key: write λA and λB equations with (2.303/20)·log(w0/w); SUBTRACT — the w0 terms cancel (equal weights) leaving the 4:1 ratio; λA known ⇒ λB ⇒ t½(B). Weight ∝ nuclei count justifies using weights.
35. **Sr-90 (t½ = 28.1 yr), 1 μg absorbed by a newborn; amount left after 20 yr** — Open: "Bahut aasaan question hai. IIT ka PYQ par bahut aasaan." Key: discard the flavor text ("pehli statement kisi kaam ki nahi"); λ = 0.693/28.1; kt = 2.303·log(w0/w) with w0 = 10⁻⁶ g, t = 20 yr ⇒ solve for w. Close: "That's simple."
