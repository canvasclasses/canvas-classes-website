# 🚀 NEET Mock Test Solution Workflow

This workflow is designed for **NEET Mock Test** questions where speed and identifying "traps" are more important than long derivations. The goal is to provide a "Whiteboard-Style" explanation that a student can read in 60 seconds to understand the most efficient path to the answer.

---

## 🚫 The "Anti-AI" Rules (Strict Directives)

1. **NO FILLER**: Do not use "Let's solve this," "We can see that," or "In conclusion." Start with the core logic immediately.
2. **NO HASHTAG HEADINGS**: Use bold text with emojis for section titles.
   - *Correct*: 🧠 **The Core Logic**
   - *Incorrect*: ### 🧠 The Core Logic
3. **CONCISED FLOW**: Keep explanations punchy. Use short sentences. Use conversational bridges like "Wait, notice the units...", "Don't fall for...", "The trick here is...".
4. **LATEX ACCURACY**: 
   - Inline math: `$ ... $`
   - Block math: `$$ ... $$`
   - Chemistry: `\ce{...}` (e.g., `\ce{KMnO4}`)

---

## 🧱 The Mandatory 3-Part Solution Structure

Every mock test solution must follow this exact 3-part sequence.

### 1. 🧠 **The Core Logic (The "Aha!" Moment)**
- **Goal**: Identify the exact concept being tested in the first 10 seconds.
- **Content**: What is the subtle hint the examiner left? Break down the complex-looking scenario into a simple physical or chemical principle. No heavy calculation here.

### 2. ⚡ **The Velocity Path (The Fastest Way)**
- **Goal**: Give the student the competitive edge.
- **Content**: Provide the most efficient calculation or logic path. Prioritize shortcuts, option elimination, or mental math over long step-by-step derivations. If there's a formula trick, lead with it.

### 3. ⚠️ **The Examiner's Trap (Avoid Negative Marks)**
- **Goal**: Identify the exact "silly mistake" or "trick" intended to catch students.
- **Content**: Speak directly to the student. Point out where they might miss a units conversion (cm to m), a sign error, or a subtle word in the question (e.g., "incorrect" vs "correct").

---

## 🎯 Final Boxed Answer
Always end with:
`$$\boxed{\text{Answer: (Option)}}$$`

---

## 📝 Example Output Template
🧠 **The Core Logic**
Briefly state the principle (e.g., conservation of angular momentum). Identify why this specific problem looks hard but is actually simple.

⚡ **The Velocity Path**
Show the "smart" calculation. 
Example: "Instead of solving the full quadratic, just check the units. Only option (b) has the dimensions of velocity."

⚠️ **The Examiner's Trap**
"Watch out for the radius vs diameter here. Most students will plug in $d$ directly and get option (a) which is exactly double the correct answer."

$$\boxed{\text{Answer: (b)}}$$
