# Free-platform keyword research — 2026-07-20

*Methodology: Google autocomplete (gl=in) + live SERP competitor mining; no paid tools. GSC figures cited where available.*

**Cross-cutting rules for the metadata-rewrite pass** (derived from all 8 families):
- **Query word order wins**: students type chapter/topic FIRST ("mole concept jee questions", "distillation diagram class 9") — front-load the topic token inside the first ~50 chars (mobile fold, ~70% of traffic).
- **Never claim "PDF"** unless a genuine downloadable PDF is served (NCERT PDF pages: yes; Live Books, tools, question hubs: no). "Free" IS honestly claimable everywhere and is a proven click trigger.
- **Year-stamp** where freshness is queried: session year "2026-27" on NCERT/notes/new-book pages; exam year on predictor/PYQ pages. Bump annually (April ritual).
- **"by Paaras Sir"** in descriptions — students search notes by teacher name (Prashant Kirad, Shobhit Nirwan, PW); build the same branded-search equity.
- **English-only titles** (site policy). Hindi demand is recorded per family + in the closing note; serve it via on-page gloss/transliteration/descriptions, never title language changes.
- Keep Indian-English demand phrasing verbatim: "time table", "rank wise", "category wise", "chapter wise", "PYQ".

## 1. Class 9 Science (`class9_science`)

### Demand themes
- **Diagram-first demand** — searchers want a drawable labelled diagram, not prose. Evidence: "tyndall effect class 9 diagram", "distillation setup diagram class 9", "simple distillation diagram class 9 ncert", "coagulation diagram class 9". GSC: "distillation diagram class 9" = **956 imps @ 0.0% CTR**. Title must literally say "diagram" (ideally "labelled diagram").
- **"Easy definition / simple words / short answer"** — "tyndall effect class 9 easy definition", "…in simple words", "…very short answer". They want a 1–3 line memorisable CBSE answer; "easy"/"simple" in title matches the query string.
- **NCERT anchoring + "which chapter" confusion** — "tyndall effect class 9 which chapter", "…chapter name", "…ncert". Name the chapter in the description ("NCERT Ch 2 — Is Matter Around Us Pure"); a bare "Chapter 5" number answers nothing.
- **Numbered examples** — "five examples of tyndall effect class 9", "give two examples of…". Teachoo wins with "What is Sublimation? - 5+ Examples with Diagram".
- **Exam-format Q&A** — "case based question on tyndall effect class 9" (×2), "is matter around us pure class 9 important questions with answers", "…mcq". Currently absorbed by Q&A aggregators.
- **Activity/experiment/animation** — "tyndall effect activity class 9", "…animation", "…project file". Canvas's simulation stack is a unique match; competitors are static PDFs.
- **Concept-pair comparisons** — "brownian motion and tyndall effect class 9", "does suspension show tyndall effect class 9", "why does colloid show tyndall effect class 9". One comparison table answers the whole cluster.
- **Eudiometry = intent mismatch** — GSC 924 imps @ 0.1% CTR, but ALL autocomplete modifiers are Class-11/JEE ("eudiometry class 11", "eudiometry jee", "eudiometry formula"). A Class-9-framed page can never win this click.
